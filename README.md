Unity Taptic Plugin
===

iOS TAPTIC Engine Bridge for Unity. 


## Test samples

Add `TapticPlugin/Samples/Sample UI` and `TapticPlugin/Sample/Sample Ball` into Build Settings/Scenes In Build.

## Required Environment

- iOS 10 or later,  
- iPhone 7 / iPhone 7 Plus or later
